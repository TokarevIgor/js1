Чему будут равны переменные x и a в примере ниже? Написать почему так произошло (описать последовательность действий).


var a = 2;
var x = 1 + (a *= 2);
Ответ: x будет = , сначала выполняется операция в скобках, т.к. приоритет выше и корректность записи, далее сумма, полная запись x = 1 + (a = a * 2 = 4) = 5.
Ответ: a будет = , a умножается на 2 то есть a = (2 * 2) = 4.
